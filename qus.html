Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer :
innerHTML: When you need to handle or manipulate HTML content, including tags.
innerText: When you want the visible text (what the user sees).
textContent: When you need raw text (ignoring tags and visibility).

Question-4: What is event bubbling in the DOM?

Answer :Event bubbling is a mechanism in the DOM where an event triggered on a child element propagates (or "bubbles up") through its ancestors, eventually reaching the root of the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer : Create an element: document.createElement("tagName") Add an element: parent.appendChild(newElement) or parent.append(newElement) Remove an element: parent.removeChild(childElement) or element.remove()
